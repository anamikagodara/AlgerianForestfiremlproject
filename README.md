# Algerian Forest Fire Prediction

## Project Overview

The Algerian Forest Fire Prediction project is a Machine Learning web application that predicts forest fire risk using environmental and weather-related parameters. The project includes data preprocessing, exploratory data analysis, model training, evaluation, and deployment using Flask.

This project demonstrates the practical implementation of Machine Learning for environmental monitoring and prediction systems.

---

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature scaling using StandardScaler
- Machine Learning model training and evaluation
- Forest fire prediction using user inputs
- Flask web application deployment
- Interactive frontend using HTML

## Dataset Information

The dataset contains meteorological and environmental attributes such as:

- Temperature
- Relative Humidity (RH)
- Wind Speed (Ws)
- Rain
- Fine Fuel Moisture Code (FFMC)
- Duff Moisture Code (DMC)
- Initial Spread Index (ISI)
- Fire Weather Index (FWI)

Target Variable:
- Fire occurrence prediction
